WinZip® позволяет легко создавать Zip-файлы, которые «распространяются» на
несколько дисков. Это полезно, если файлы, с которыми вы работаете, не помещаются
на один диск.
Создание составных Zip-файлов
Чтобы создать Zip-файл, который охватывает несколько дисков, просто создайте
новый Zip-файл на съемном диске, а затем добавьте файлы в Zip, как обычно. Если
диск заполняется во время операции добавления, вам будет предложено ввести
другой диск. Просто вставьте следующий диск и нажмите кнопку OK.
Важно: охват диска недоступен при добавлении файлов в существующий архив; он
доступен только при создании нового архива. Также обратите внимание, что вы не
можете добавлять файлы или удалять файлы из Zip-файла, который охватывает
диски. Для WinZip требуются предварительно отформатированные диски (вы можете
отформатировать диски с помощью «Мой компьютер» или проводника Windows).
Пример :
Чтобы заархивировать файлы в папке c:\data в Zip-файл с именем A:\DATA.ZIP:
Запустите WinZip.
Перетащите папку c:\data из папки «Мой компьютер» или проводника Windows в окно
WinZip.
В диалоговом окне «Добавить» введите имя файла Zip (A:\DATA.ZIP).
Нажмите кнопку «Добавить».
Если требуется более одного диска, вам будет предложено установить
дополнительные диски.
Извлечение из составных файлов Zip
Чтобы извлечь файлы из составного Zip-файла, вставьте последний диск серии и
откройте Zip-файл с помощью WinZip; это отобразит каталог Zip-файла и позволит вам
извлечь файлы. WinZip запросит у вас диски по мере необходимости для завершения
операции.
