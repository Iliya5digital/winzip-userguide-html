Вот пошаговые инструкции по установке ARJ, дополнительной внешней программы,
необходимой для работы с файлами ARJ.
ARJ распространяется в виде самораспаковывающегося архива с именем ARJ n .EXE,
где n — номер версии. Обратите внимание, что WinZip поддерживает только
16-разрядные версии ARJ и не поддерживает 32-разрядные версии ARJ. Самая ранняя
версия ARJ, поддерживаемая WinZip®, — ARJ 2.50a (ARJ250A.EXE); для получения
текущей версии ARJ, поддерживаемой WinZip, посетите
http://www.winzip.com/xextern.htm . Сначала загрузите файл с вашего любимого
онлайн-сервиса.
Используйте Мой компьютер или Проводник Windows, чтобы создать папку C:\ARJ. Вы
можете использовать другую папку, но тогда вам нужно будет заменить это имя папки в
следующих инструкциях.
Используйте «Мой компьютер» или проводник Windows, чтобы скопировать
исполняемый файл установки ARJ из папки загрузки в папку C:\ARJ.
Дважды щелкните файл, чтобы запустить его. Это заставит файлы
«самораспаковываться» в папку C:\ARJ. Затем вы можете удалить исполняемый файл
установки, если хотите, так как он не нужен для повседневной работы с ARJ и WinZip.
Запустите WinZip. Откройте меню « Параметры » и нажмите « Конфигурация » , затем
перейдите на вкладку « Расположение программ ».
В поле ARJ вкладки Program Locations диалогового окна Configuration введите
c:\arj\arj.exe
и нажмите кнопку OK .
Теперь вы можете получить доступ к файлам ARJ так же, как к файлам Zip.
Примечание: как и WinZip, ARJ не является бесплатным программным обеспечением.
Как и в случае со всеми условно-бесплатными программами, если вы используете эти
программы, вы должны зарегистрировать их у соответствующего поставщика.
Подробности смотрите в файле license.doc в папке ARJ.
Примечание. ARJ 2.60 и более поздние 16-разрядные версии обеспечивают поддержку
длинных имен файлов в Windows 95, Windows 98 и Windows Me. Поддержка длинных
имен файлов недоступна при работе 16-разрядных версий ARJ под Windows NT 4.0,
Windows 2000 или Windows XP.
