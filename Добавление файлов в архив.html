Новые пользователи: вводную информацию см. в разделе Краткое руководство —
добавление файлов в архив .
В этом разделе описывается диалоговое окно «Добавить». Есть и другие способы
добавления файлов в архив:
Интерфейс перетаскивания .
Интерфейс проводника .
Диалоговое окно «Добавить»
Чтобы открыть диалоговое окно «Добавить», нажмите кнопку « Добавить » на панели
инструментов или выберите пункт « Добавить » в раскрывающемся меню « Действия
», чтобы открыть диалоговое окно «Добавить».
Вы можете выбрать файлы, которые хотите добавить, несколькими способами:
Нажмите на нужные файлы в списке в верхней части диалогового окна «Добавить». Вы
можете выбрать несколько файлов, используя обычные правила выбора файлов
Windows (например, удерживая нажатой клавишу Shift, щелкните, чтобы выбрать
несколько файлов). Затем нажмите кнопку Добавить .
Введите имена файлов, которые вы хотите добавить. Если вы указываете несколько
имен файлов, каждое имя файла должно быть заключено в двойные кавычки. Затем
нажмите кнопку Добавить .
Чтобы добавить файлы с подстановочными знаками (например, «*.doc», чтобы
добавить все файлы с расширением .doc), введите имя файла с подстановочными
знаками в поле редактирования имени файла , затем нажмите кнопку « Добавить с
подстановочными знаками » . Принимаются стандартные подстановочные знаки.
Описание параметров, доступных в этом диалоговом окне, см. в разделе «Добавить
параметры диалогового окна».
Примечание о совместимости
Начиная с версии 9.0, WinZip® поддерживает 64-разрядные расширения формата
файлов Zip. Расширенный формат позволяет хранить все необходимые данные в
Zip-файлах практически неограниченного размера.
Исходный формат файла Zip ограничивал количество файлов-членов в файле Zip до
65 535, а максимальный размер как самого файла Zip, так и любого файла-члена - до 4
гигабайт. Для всех практических целей 64-битный расширенный формат устраняет все
эти ограничения: размер файла-члена, размер файла Zip и количество файлов-членов
ограничены только ресурсами вашей системы.
WinZip остается полностью совместимым с исходным форматом файла и по
возможности использует исходный формат. WinZip использует 64-битный расширенный
формат только тогда, когда превышены пределы исходного формата.
Однако следует помнить, что очень большие Zip-файлы, сжатые с помощью WinZip
9.0, несовместимы с более ранними версиями WinZip и могут быть несовместимы с
другими утилитами Zip-файлов.
