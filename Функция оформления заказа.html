Функция CheckOut упрощает проверку и/или запуск файлов в архиве. Он создает
временную папку, извлекает все файлы из архива в папку, создает записи в меню
«Программы» для извлеченных элементов и открывает окно в стиле «Мой компьютер»,
показывающее ярлыки для файлов в папке. Дважды щелкните любой элемент в окне,
чтобы выполнить действие, обычно связанное с документом или программой (связи
описаны в разделе «Связи Windows» ). WinZip® при желании закроет окно и удалит
временную папку (и извлеченные файлы), когда вы закроете архив или закроете
WinZip.
Диалоговое окно «Оформить заказ»
Используйте кнопку CheckOut на панели инструментов или пункт CheckOut в
раскрывающемся меню Actions , чтобы активировать диалоговое окно CheckOut. Это
диалоговое окно позволяет настроить следующие аспекты операции:
Используйте поле редактирования Папка , чтобы указать имя временной папки
CheckOut. Папка будет создана, если она еще не существует. Предлагаемая папка
зависит от настройки базовой папки CheckOut на вкладке « Папки » диалогового окна
«Конфигурация».
Используйте поле редактирования Имя группы , чтобы указать имя, которое будет
использоваться для окна CheckOut.
Используйте поле редактирования Максимальное количество значков , чтобы указать
максимальное количество элементов, которые будут помещены в окно CheckOut.
Флажок Создавать значки только для программ и документов позволяет указать, будет
ли WinZip создавать значки для всех файлов в архиве или только для указанных
файлов. При выборе этого параметра иконки создаются для исполняемых файлов и
файлов с ассоциациями . Иконки также создаются для любых файлов с именами,
начинающимися с «read» (чтобы отлавливать файлы с именами типа «readme.1st»,
«read.me» и т. д.).
Флажок Запустить антивирусный сканер позволяет указать, будет ли WinZip запускать
антивирусный сканер для файлов в архиве. Этот параметр отключен, если на вкладке
« Расположение программ » диалогового окна «Конфигурация» не установлена
утилита сканирования на вирусы .
Операции, выполняемые функцией CheckOut:
Когда вы нажимаете кнопку OK в диалоговом окне CheckOut, WinZip выполняет
следующие операции:
Если указанная папка еще не существует, WinZip создаст ее. Если он уже существует и
содержит файлы, WinZip удалит файлы (конечно, спросив у вас разрешения). Это
гарантирует, что папка будет содержать только файлы из текущего архива.
WinZip извлекает файлы из текущего архива в указанную папку.
Если установлен флажок Запустить антивирусный сканер , ваш антивирусный сканер
запускается для всех файлов в указанной папке. Если утилита поиска вирусов
возвращает ненулевой уровень ошибки, операция CheckOut отменяется и
отображаются сообщения от сканера.
WinZip создает вторую папку, расположенную в \Windows\Start Menu\Programs. Если
папка с указанным именем уже существует, WinZip удалит ее после получения вашего
разрешения. Это гарантирует, что папка будет содержать только значки для файлов в
текущем архиве.
Во второй папке WinZip создает ярлыки для извлеченных файлов. Затем он открывает
окно в стиле «Мой компьютер», в котором отображаются значки извлеченных файлов.
Эти элементы также появятся в меню «Программы», доступ к которому можно
получить с помощью кнопки «Пуск».
Использование окна CheckOut:
Окно, созданное CheckOut, представляет собой представление папки проверки в стиле
«Мой компьютер» и работает как любое подобное окно. WinZip создает ярлыки в этой
папке для каждого извлеченного файла, чтобы вы могли использовать их следующим
образом:
Если файл является исполняемым, двойной щелчок по значку запускает программу.
Если файл имеет ассоциацию, двойной щелчок по значку загружает файл в связанную
программу, чтобы вы могли просмотреть файл.
В противном случае двойной щелчок по значку загружает файл в программу
сопоставления WinZip по умолчанию , указанную на вкладке « Расположения программ
» диалогового окна «Конфигурация».
