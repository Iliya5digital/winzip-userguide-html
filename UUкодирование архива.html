Используйте запись UUencode в раскрывающемся меню Действия для UUencode
открытого архива. Файл UUencoded можно отправлять по почтовым системам, которые
не поддерживают двоичную электронную почту. Дополнительные сведения о файлах
UUencoded, XXencoded, BinHex и MIME см. в соответствующем разделе этого файла
справки.
