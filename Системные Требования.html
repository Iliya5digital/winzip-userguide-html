Системные Требования
WinZip® требует Windows 98, Windows Me, Windows XP, Windows NT 4.0 или Windows
2000. Он не будет работать под Windows 3.x или Windows NT 3.1 или 3.5.
Никакие другие программы не требуются для добавления или извлечения файлов из
Zip-файлов. Однако для некоторых необязательных операций WinZip требуются
внешние программы или файлы:
Дополнительная поддержка сканирования на вирусы требует, чтобы в вашей системе
была установлена поддерживаемая программа сканирования на вирусы. См. ниже
конкретную информацию и информацию о том, как найти список антивирусных
сканеров, автоматически обнаруживаемых WinZip.
Для дополнительной поддержки некоторых старых форматов (ARC, ARJ и LHA)
требуется одна из поддерживаемых программ, перечисленных ниже. Пошаговые
инструкции по установке этих программ см. в разделе « Установка дополнительных
внешних программ » .
Вот более подробная информация о некоторых дополнительных программах:
Вы можете создавать самораспаковывающиеся Zip-файлы с помощью WinZip
Self-Extractor Personal Edition , входящего в состав WinZip, или вы можете использовать
полную версию WinZip Self-Extractor , сопутствующий продукт за 49 долларов, который
включает в себя функции, адаптированные для распространения программного
обеспечения, и возможность создания единого архива. самораспаковывающийся
Zip-файл, который можно запускать под Windows.
Для доступа к файлам LZH требуется программа LHA.EXE. Для получения последней
информации о получении LHA.EXE посетите веб-сайт WinZip по адресу
http://www.winzip.com/xextern.htm . WinZip не будет работать с версиями ранее LHA 2.13
(в частности, он не будет работать с LHARC, предшественником LHA).
Для доступа к файлам ARJ требуется программа ARJ от Роберта Юнга. Вы можете
загрузить условно-бесплатную версию ARJ из Интернета и основных онлайн-сервисов.
Чтобы получить последнюю версию, поддерживаемую WinZip, посетите веб-сайт
WinZip по адресу http://www.winzip.com/xextern.htm . WinZip не будет работать с
версиями ранее ARJ 2.5 (в частности, он не будет работать с ARJ32).
Для доступа к файлам ARC требуется одна из следующих программ:
ARCE.COM и ARC-E.COM версии 4.0e — утилита Vern Buerg для извлечения данных
из ARC, доступная в Интернете и основных онлайн-сервисах. WinZip не тестировался с
более ранними версиями.
PKXARC.COM версии 3.5 от PKWARE, Inc. WinZip не тестировался с более ранними
версиями. Эта программа все еще может быть доступна в некоторых местах в
Интернете.
PKXARC.EXE версии 3.6 от PKWARE, Inc. Эта программа все еще может быть
доступна в некоторых местах в Интернете.
PKUNPAK.EXE версии 3.61 от PKWARE, Inc. Эта программа все еще может быть
доступна в некоторых местах в Интернете.
ARC.EXE версии 5.20 или 6.0 от System Enhancement Associates, Inc. Это последняя
версия ARC, доступная как условно-бесплатная, и ее все еще можно найти в
некоторых местах в Интернете. WinZip не тестировался с более ранними версиями.
WinZip автоматически обнаружит несколько популярных антивирусных сканеров;
текущий список можно найти на веб-сайте WinZip по адресу
http://www.winzip.com/xvirus.htm .
WinZip также можно настроить для работы с большинством утилит для сканирования
на вирусы. Важные подробности см. в разделе Конфигурация сканера вирусов .
Для некоторых параметров конфигурации панели инструментов требуется версия 4.71
или более поздняя библиотека общих элементов управления Windows. Эта библиотека
является частью Windows 98, Windows Me, Windows 2000, Windows XP и Microsoft
Internet Explorer 4.0 или более поздней версии и доступна для отдельной загрузки с
веб-сайта Microsoft, как описано в разделе Общая библиотека управления Windows .
