WinZip® 9.0 SR-1 — это отладочная версия WinZip 9.0. Эта версия WinZip была
улучшена, чтобы предупредить вас о некоторых наиболее распространенных
ситуациях, в которых вы можете заразиться вирусом, сжатым внутри Zip-файла.
Полную информацию об этом изменении см . в разделе « Потенциально небезопасные
типы файлов» .
Основные изменения в WinZip 9.0
Улучшения в WinZip 9.0 сосредоточены на его основных функциях: сжатии, емкости и
новой расширенной возможности шифрования данных. Используя WinZip 9.0, вы
можете сжимать больше данных, сжимать их лучше и защищать конфиденциальную
информацию с гораздо большей безопасностью.
Подробная информация об этих и других улучшениях приведена ниже.
Расширенное шифрование
WinZip 9.0 поддерживает 128- и 256-битное шифрование AES, что обеспечивает
гораздо большую криптографическую безопасность, чем традиционный метод
шифрования Zip 2.0, используемый в более ранних версиях WinZip.
Расширенное шифрование WinZip 9.0 (сертифицировано по FIPS-197) использует
криптографический алгоритм Rijndael, который в 2001 году был указан Национальным
институтом стандартов и технологий (NIST) в публикации 197 Федеральных
стандартов обработки информации (FIPS) в качестве расширенного стандарта
шифрования. (АЭС).
После трехлетнего конкурса NIST объявил AES одобренным методом шифрования для
использования правительством США, частными предприятиями и частными лицами.
При правильной реализации в качестве ключевого компонента общего протокола
безопасности AES обеспечивает очень высокую степень криптографической защиты,
но при этом работает быстро и эффективно.
Шифрование WinZip AES так же просто в использовании, как и традиционное
шифрование Zip 2.0: все, что вам нужно сделать, это выбрать уровень шифрования и
указать свой пароль.
Примечание: получатели, которым вы отправляете Zip-файлы, зашифрованные с
помощью AES, должны иметь совместимую утилиту Zip-файлов для расшифровки
файлов. Мы опубликовали полную спецификацию для создания WinZip-совместимых
Zip-файлов с шифрованием AES и ожидаем, что другие поставщики Zip-утилит
обеспечат поддержку этого формата.
Большая емкость
Помимо поддержки исходного формата файлов Zip, WinZip 9.0 также поддерживает
64-разрядные расширения формата файлов Zip. Расширенный формат позволяет
хранить все необходимые данные в Zip-файлах практически неограниченного размера.
Исходный формат файла Zip ограничивал количество файлов-членов в файле Zip до
65 535, а максимальный размер как самого файла Zip, так и любого файла-члена - до 4
гигабайт. Для всех практических целей 64-битный расширенный формат устраняет все
эти ограничения. При использовании расширенного формата размер файла участника,
размер файла Zip и количество файлов участников, которые вы можете добавить в
файл Zip, ограничены только ресурсами вашей системы.
WinZip остается полностью совместимым с исходным форматом файла и по
возможности использует исходный формат. WinZip использует 64-битный расширенный
формат только тогда, когда превышены пределы исходного формата.
Улучшенное сжатие
WinZip 9.0 поддерживает метод сжатия «улучшенное сжатие». Этот метод сжатия
обеспечивает лучшее сжатие для многих типов файлов и уменьшает размер
Zip-файлов, экономя время на передачу данных и драгоценное место на диске.
Примечание: версии WinZip до WinZip 8.1 не смогут извлекать файлы, сжатые с
помощью этого нового метода.
Другие изменения в WinZip 9.0
В дополнение к новой технологии шифрования AES WinZip 9.0 предоставляет ряд
улучшений удобства использования, упрощающих использование шифрования.
Например, теперь вы можете легко зашифровать файлы, которые уже находятся в
Zip-файле; ранее файлы можно было зашифровать только во время их добавления в
Zip-файл.
WinZip 9.0 упрощает поиск необходимой информации WinZip, предоставляя
справочную информацию с помощью нового средства Microsoft «HTML Help».
HTML-справка имеет более привлекательный вид и включает множество улучшений
удобства использования.
WinZip 9.0 упрощает доступ к наиболее важным расположениям файлов во многих
диалоговых окнах WinZip Classic. Ключевые диалоговые окна, такие как «Новый
архив», «Открыть архив», «Добавить» и «Извлечь», содержат «полосы мест» при
работе в версиях Windows, которые их поддерживают. Панель мест обеспечивает
быстрый доступ к местам верхнего уровня, обычно к моему компьютеру, моей сети
(или сетевому окружению), моим документам, последним документам (или истории) и
рабочему столу.
WinZip 9.0 позволяет быстро узнать, что находится в ваших Zip-файлах. В «Моем
компьютере» и проводнике Windows всплывающая подсказка, отображаемая при
наведении указателя мыши на файл Zip, теперь содержит информацию о содержимом
ваших архивов, перечисляя до 25 файлов и папок в архиве.
При установке в Windows XP WinZip 9.0 устанавливается и регистрируется для всех
пользователей. Это изменение упрощает установку WinZip в рабочих или домашних
условиях, когда компьютером пользуются несколько человек.
WinZip 9.0 содержит ряд изменений для повышения производительности и
эффективности:
Параметр «Использовать только для съемных носителей» для рабочей папки (на
вкладке «Папки» диалогового окна «Конфигурация») теперь установлен по умолчанию.
В большинстве случаев это приводит к более быстрому созданию Zip-файла.
По умолчанию WinZip больше не отображает автоматически комментарии при
открытии Zip-файлов, содержащих комментарии.
В сетевых средах Windows XP улучшена производительность расширений Windows
Explorer WinZip.
Внесено несколько изменений для повышения производительности при извлечении
файлов из архивов.
Улучшен классический интерфейс WinZip 9.0:
Диалоговое окно «Извлечение» было переработано, чтобы отображать больше папок в
представлении дерева папок и предоставлять больше места для ввода имен папок.
При наведении указателя мыши на усеченное имя файла (которое не помещается на
доступном месте) отображается всплывающая подсказка, показывающая полное имя
файла.
Поскольку это действие, а не настройка, параметр «Восстановить все
предупреждающие сообщения» на вкладке «Разное» диалогового окна
«Конфигурация» был изменен с флажка на кнопку.
Главное окно WinZip теперь содержит водяной знак при использовании в Windows XP с
темой в стиле XP.
WinZip 9.0 больше не выдает ложные предупреждения в некоторых инструментах
безопасности, которые ищут программы, использующие библиотеку сжатия «zlib». zlib
не является компонентом WinZip.
