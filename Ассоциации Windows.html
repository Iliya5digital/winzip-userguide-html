Ассоциации Windows
Microsoft определила слово «ассоциировать» следующим образом:
Для идентификации расширения имени файла как принадлежащего определенному
приложению, чтобы при открытии любого файла с этим расширением правильное
приложение открывалось автоматически.
Например, по умолчанию Windows связывает файлы .WRI с приложением Write или
WordPad, а файлы .TXT — с приложением NOTEPAD. Большинство процедур
установки приложений Windows создают ассоциации. Например, Word для Windows
создает ассоциацию для файлов .DOC, тогда как WinZip® обычно создает ассоциации
для файлов с расширениями ZIP, LZH, ARJ, ARC, CAB, TAR, TGZ, TAZ, TZ, GZ, Z, UU,
UUE. , ХХЕ, B64, HQX, BHX, МИМ.
WinZip использует ассоциации, когда вы дважды щелкаете имя файла, указанное в
главном окне WinZip, и когда вы используете функцию извлечения .
WinZip, My Computer и Windows Explorer используют ассоциации одинаковым образом.
Когда вы открываете файл с ассоциацией, запускается приложение, связанное с
файлом, и приложение автоматически открывает файл. Средство WinZip CheckOut
устанавливает ассоциации при создании окна CheckOut по двум причинам: для
отображения правильного значка для каждого файла и для обеспечения запуска
соответствующего приложения при двойном щелчке значка.
Вы можете просматривать, добавлять и удалять ассоциации Windows с помощью «Мой
компьютер» или проводника Windows. Откройте «Мой компьютер» или «Проводник
Windows» и найдите в меню « Параметры » или «Свойства папки » (точное название и
расположение меню зависит от вашей версии Windows). Откройте диалоговое окно
параметров из меню и щелкните вкладку Типы файлов . Нажмите клавишу F1 для
получения подробных инструкций.
Ассоциации WinZip
Вы можете легко просматривать и изменять типы архивов, с которыми WinZip связан,
из WinZip. Когда тип архива связан с WinZip, Windows будет использовать WinZip для
открытия файлов этого типа.
Чтобы просмотреть и изменить ассоциации WinZip, выберите « Конфигурация » в
меню « Параметры », перейдите на вкладку « Система » и нажмите кнопку «
Ассоциации» . В следующем диалоговом окне вы увидите список типов архивов. Вы
можете указать, должен ли тип архива быть связан с WinZip, установив или сняв
флажок рядом с типом архива.
