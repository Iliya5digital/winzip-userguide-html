WinZip® поддерживает множество антивирусных сканеров, которые можно запускать
из Windows или из командной строки DOS, с некоторыми важными ограничениями,
указанными ниже. Сканеры вирусов можно запускать с помощью пункта «
Сканирование вирусов» в меню «Действия» или во время операций извлечения, если
выбран параметр « Запустить сканер вирусов » в диалоговом окне «Извлечение».
Настройка антивирусного сканера выполняется на вкладке Program Locations
диалогового окна Configuration.
Когда вы впервые устанавливаете WinZip, он автоматически ищет антивирусные
сканеры в папках, указанных в PATH вашей системы, и в записях реестра Windows
«Пути приложений». Если обнаружен один из поддерживаемых антивирусных
сканеров, WinZip автоматически использует этот сканер. Список и дополнительную
информацию о поддерживаемых в настоящее время сканерах вирусов можно найти на
веб-сайте WinZip по адресу http://www.winzip.com/xvirus.htm .
Если процедура установки не находит автоматически один из вышеупомянутых
антивирусных сканеров, вы можете сделать так, чтобы WinZip автоматически заполнил
поле « Параметры », выбрав нужный сканер из раскрывающегося списка (например,
scan.exe), а затем добавив полный имя файла сканера в поле Программа
сканирования (например, c:\mcafee\scan.exe вместо просто scan.exe).
Вы также можете настроить WinZip для работы с другим сканером. Дополнительные
сведения см. в разделе « Сведения о конфигурации сканера вирусов» ниже.
Заметки:
Параметры антивирусного сканера WinZip по умолчанию работают с
протестированными версиями сканеров. Если авторы сканера изменят параметры
командной строки в будущих выпусках сканера, то параметры WinZip по умолчанию
для сканера могут больше не работать. Например, параметры McAfee SCAN 2.1
несовместимы с предыдущими версиями SCAN. Для версии 2 требуется параметр
/ALL, а для версии 1 — параметр /A.
Если вирус не обнаружен, при работе большинства антивирусных сканеров ручное
вмешательство не требуется. Однако некоторые антивирусные сканеры, такие как
TFINDVIR, всегда необходимо закрывать вручную, поскольку в настоящее время они
не поддерживают параметры автоматического закрытия.
Сведения о конфигурации сканера вирусов
В этой части документации описывается, как настроить WinZip для работы со
сканером, не указанным выше. Не используйте поддержку сканирования вирусов
WinZip с другим антивирусным сканером, если вы не прочитали и не поняли
следующую информацию.
Чтобы настроить WinZip для работы со сканером, заполните следующие поля на
вкладке Program Locations диалогового окна Configuration:
Поле Программа сканирования должно содержать имя программы (полный путь
требуется, если программа не находится в папке, указанной в PATH вашей системы).
Поле « Параметры » должно содержать соответствующие параметры для запуска
антивирусного сканера, как описано ниже.
Поле Запустить в свернутом виде определяет, следует ли запускать сканер в
свернутом виде. Ниже приведены важные примечания по использованию этого
параметра со сканерами на базе DOS. Обратите внимание, что некоторые сканеры,
такие как NAVW32.EXE, всегда работают в свернутом виде, независимо от того, как вы
установили этот параметр.
Специальные последовательности символов для поля параметров :
*.*
указывает, где WinZip должен заменить диск, папку и символы *.* для файлов,
подлежащих сканированию. Например, "c:\temp\dir\*.*".
%d
указывает, где WinZip должен заменить диск и папку с файлами, которые должны быть
просканированы. Обратите внимание, что подстановочные знаки *.* НЕ включаются
при использовании %d.
%f
указывает, где WinZip должен заменить имя временного «файла отчета». Файлы
отчетов следует использовать ТОЛЬКО при запуске антивирусных сканеров DOS.
%wscan
специальный параметр, необходимый для программы McAfee WSCAN.
Если вы не укажете какую-либо из вышеупомянутых последовательностей
специальных символов, WinZip добавит имена файлов для сканирования в конце
командной строки сканера.
Обязательно укажите любые параметры, необходимые для того, чтобы сканер
проверял папки диска и папки с файлами, которые должны быть просканированы.
Чтобы настроить WinZip для антивирусного сканера на базе Windows, сначала
убедитесь, что антивирусный сканер сообщает о подозрительных вирусах, отображая
диалоговое окно. Если сканер просто записывает в лог-файл при обнаружении вируса,
вы (и WinZip) не будете знать о проблеме.
Чтобы настроить WinZip для сканера вирусов на основе DOS, обязательно убедитесь,
что сканер можно использовать с параметром « Запустить в свернутом виде»
(подробности см. ниже) или выдает запрос при обнаружении вируса. Если ни одно из
этих условий не выполнено и обнаружен вирус, вы (и WinZip) не узнаете о проблеме.
Флажок Запустить в свернутом виде следует использовать только для сканеров DOS
при следующих обстоятельствах:
Сканер настроен на работу в "пакетном" режиме без каких-либо подсказок. Если
сканеру требуется какой-либо ввод с клавиатуры при работе в свернутом виде, вы не
сможете увидеть подсказки, и сеанс будет выглядеть зависшим.
Сканер записывает все сообщения через DOS или имеет возможность записывать
результаты в файл отчета. Сообщения, записанные непосредственно в видеобуфер,
не захватываются WinZip и будут потеряны. Чтобы определить, пишет ли сканер
сообщения через DOS, запустите сканер с перенаправленным выводом (например,
введите команду «scan > nul»). Любые сообщения, которые вы увидите во время этого
теста, не будут захвачены WinZip. Если у сканера есть возможность записывать
результаты в файл отчета, вы можете использовать %f в поле « Параметры » , чтобы
WinZip зафиксировал результаты (подробности см. выше).
Сканер возвращает ненулевой уровень ошибки, если обнаруживает какие-либо
проблемы. Если сканер не возвращает ненулевой уровень ошибки, WinZip будет
считать, что файлы прошли проверку на наличие вирусов, и все предупреждения,
выданные сканером, будут потеряны. (Примечание: если сканер возвращает
ненулевой уровень ошибки, WinZip автоматически отображает все сообщения,
записанные в стандартную ошибку или стандартный вывод.)
