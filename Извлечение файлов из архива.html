Извлечение файлов .
В этом разделе описывается диалоговое окно Извлечь. Есть и другие способы
извлечения файлов из архива:
Использование интерфейса перетаскивания .
Непосредственно с Моего компьютера или Проводника Windows, как описано в
разделе Использование интерфейса Проводника .
Если Zip-файл содержит программу установки или установки, вы можете использовать
функцию установки для автоматического извлечения файлов, запуска программы
установки и очистки временных файлов.
Диалоговое окно «Извлечение»
Используйте кнопку « Извлечь » на панели инструментов или выберите « Извлечь » в
меню « Действия », чтобы открыть диалоговое окно «Извлечь». Используйте
диалоговое окно «Извлечь», чтобы извлечь или «разархивировать» файлы из
открытого архива на диск.
WinZip® извлекает файлы из архива в папку, указанную в поле Извлечь в . Вы можете
указать эту папку одним из четырех способов:
Выберите папку и диск в списках Папки/диски .
Введите имя папки в поле Извлечь в .
Раскройте поле со списком « Извлечь в » и выберите папку из списка предыдущих
папок « Извлечь в » .
Нажмите кнопку «Новая папка» изображение\НоваяПапка.gifи создайте папку.
В диалоговом окне Извлечь доступны следующие параметры:
Кнопка выбора Выбранные файлы/папки указывает WinZip извлекать только те файлы,
которые выбраны в главном окне WinZip. Для выбора файлов можно использовать
обычные методы выбора файлов. Эта опция отключена, если файлы не выбраны.
Кнопка « Все файлы/папки в архиве » указывает WinZip извлечь все файлы из архива.
Кнопка выбора « Файлы » и поле редактирования позволяют указать, какие файлы
извлекать, используя имена файлов (допускаются стандартные подстановочные знаки
Windows, поэтому вы можете использовать «*.doc» для извлечения всех файлов .doc)
Флажок Перезаписать существующие файлы определяет, будет ли отображаться
запрос перед перезаписью существующих файлов файлами из архива.
Флажок Пропускать старые файлы определяет, будет ли WinZip пропускать (не
извлекать) существующие файлы, если отметка даты/времени файла в архиве старше,
чем файл на диске.
Флажок Использовать имена папок определяет, будет ли WinZip использовать имена
папок (также называемые путями), хранящиеся в архиве. Вы, вероятно, захотите
установить этот флажок для большинства операций извлечения. Если этот параметр
отмечен, любые имена папок, хранящиеся в архиве, будут добавляться к папке «
Извлечь в папку», когда это необходимо. Если этот параметр не отмечен, все файлы
будут извлечены в одну и ту же папку, независимо от имен папок в архиве .
Например, если для параметра Извлечь в папку указано «c:\temp», файл
«mydir\filename» будет извлечен в папку «c:\temp\mydir», если установлен флажок «
Использовать имена папок», или в папку « c:\temp", если опция не отмечена.
Если установлен флажок « Открыть окно проводника», после завершения операции
извлечения открывается окно проводника Windows. В окне будет указана папка, в
которую были извлечены файлы. Этот флажок отсутствует, если вы извлекаете из
нескольких Zip-файлов непосредственно из проводника Windows с помощью
контекстного меню WinZip.
Если вы извлекаете из нескольких Zip-файлов с помощью пункта контекстного меню
WinZip « Извлечь в... », появится флажок « Использовать имена Zip-файлов для папок
» . Если этот флажок установлен, WinZip создаст папку для каждого Zip-файла в папке
« Извлечь в папку»; папки будут названы базовыми именами Zip-файлов. Например,
если для параметра « Извлечь в папку» указано «c:\temp», WinZip извлечет файлы из
«NewDocs.zip» в «c:\temp\NewDocs». Если этот флажок не установлен , файлы будут
извлечены в папку Извлечь в . В большинстве случаев вы, вероятно, захотите оставить
этот параметр включенным при извлечении из нескольких Zip-файлов.
Кнопка «Новая папка» изображение\НоваяПапка.gifактивирует диалоговое окно «
Создать папку ». Обратите внимание, что вы можете извлечь файлы и создать папку
без этого диалогового окна: просто введите имя новой папки в поле « Извлечь в»
диалогового окна «Извлечь» и нажмите кнопку « Извлечь » .
Примечание. Вы можете изменить извлечение по умолчанию в папку с помощью
вкладки « Папки » диалогового окна «Конфигурация».
