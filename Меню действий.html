Используйте меню « Действия » для команд, влияющих на открытый в данный момент
архив или выбранные файлы в архиве. Меню содержит следующие пункты:
Добавить добавляет новые файлы в архив или обновляет файлы, которые уже
находятся в архиве.
Удалить удаляет файлы из архива.
Извлечь распаковывает файлы в архиве и сохраняет несжатые файлы на диск.
Просмотр позволяет просмотреть содержимое файлов в архиве.
Select All выбирает все файлы в архиве.
Инвертировать выбор выбирает все файлы, которые в данный момент не выбраны, и
отменяет выбор всех файлов, которые в данный момент выбраны.
Virus Scan использует ваш внешний антивирусный сканер для проверки файлов в
архиве на наличие вирусов.
Make .Exe File преобразует архив в самораспаковывающийся архив.
UUEncode кодирует архив, чтобы его можно было отправить через почтовые системы,
которые не поддерживают двоичную электронную почту.
Разделить — архив разбивается на несколько меньших частей указанного размера.
Шифровать
позволяет зашифровать или повторно зашифровать все файлы в Zip-файле. Это
действие доступно только для файлов Zip.
Test выполняет внутреннее тестирование, чтобы убедиться, что архив не поврежден.
Комментарий позволяет добавлять или редактировать комментарий к файлу Zip.
CheckOut упрощает проверку содержимого архива, извлекая файлы во временную
папку и настраивая окно проводника Windows для отображения файлов.
Install помогает вам устанавливать программное обеспечение из архивов, содержащих
программы SETUP или INSTALL.
