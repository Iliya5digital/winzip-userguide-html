Меню « Параметры » позволяет установить множество параметров WinZip®. Пункты
меню:
Конфигурация активирует главное диалоговое окно Конфигурация. Это диалоговое
окно управляет большинством параметров WinZip.
Сортировка позволяет указать, сортируется ли список файлов в архиве по имени, типу,
дате, размеру, коэффициенту сжатия, размеру упаковки, CRC, атрибутам файла, пути
или следует использовать исходный порядок архива. Примечание. Вы также можете
изменить порядок сортировки, щелкнув соответствующую часть заголовка основного
списка. Например, нажмите на слово «Дата» в заголовке, чтобы отсортировать по
дате. Чтобы изменить порядок сортировки, щелкните его еще раз.
Повторное использование WinZip Windows определяет, будет ли открываться новое
окно при запуске WinZip двойным щелчком в окне «Мой компьютер», проводнике
Windows или на рабочем столе. Когда эта запись отмечена флажком, используется
существующее окно WinZip. Если эта запись не отмечена, создается новое окно
WinZip. Одним из преимуществ повторного использования окон WinZip является то, что
двойной щелчок действует так же, как перетаскивание, и вы никогда случайно не
столкнетесь с несколькими окнами WinZip.
Параметр «Сохранить настройки при выходе» определяет, будут ли автоматически
сохраняться выбранные вами параметры конфигурации при закрытии WinZip.
Save Settings Now немедленно сохраняет выбранные параметры.
Set Installation Defaults устанавливает для многих параметров конфигурации и имен
папок исходные значения по умолчанию при установке.
View Last Output отображает результаты последней операции WinZip.
