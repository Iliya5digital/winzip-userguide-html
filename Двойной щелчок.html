Двойной щелчок по имени файла, указанному в главном окне WinZip®, приведет к тому
же действию, что и двойной щелчок по файлу в «Моем компьютере» или проводнике
Windows. Если файл является исполняемым (имеет расширение EXE, COM, BAT или
PIF), WinZip запустит его. В противном случае WinZip откроет файл в соответствующем
приложении. Например, файлы с расширением имени файла .TXT открываются
приложением Windows Notepad (если вы не изменили связь TXT). Подходящее
приложение определяется с помощью ассоциаций Windows . Если с файлом не
связано ни одно приложение, WinZip запустит программу, указанную в качестве
ассоциации по умолчанию в расположении программ .вкладку диалогового окна
конфигурации. Если вы модифицируете файл во время его просмотра, WinZip
предложит добавить измененный файл в архив.
Если включена опция обработки файлов Smart DOC на вкладке « Разное »
диалогового окна «Конфигурация», и вы дважды щелкните файл с типом файла .DOC,
WinZip запустит одну из двух программ:
Если файл .DOC выглядит как обычный текст ASCII, WinZip запустит программу,
указанную в качестве ассоциации по умолчанию на вкладке « Расположения программ
» диалогового окна «Конфигурация».
Если файл .DOC выглядит как файл, созданный текстовым процессором, WinZip
запустит программу, связанную с файлами .DOC.
Интеллектуальная обработка файлов DOC позволяет избежать накладных расходов,
связанных с загрузкой текстового процессора, такого как Microsoft Word, при просмотре
текстовых файлов ASCII, таких как «readme.doc». Обратите внимание, что обработка
файлов Smart DOC используется только для файлов, дважды щелкнутых; он не
используется, когда вы щелкаете правой кнопкой мыши файл .DOC и выбираете
«Открыть» в контекстном меню. В этом случае WinZip использует для открытия
документа ту же программу, что и проводник Windows; для большинства
пользователей это будет Microsoft Word или другой текстовый процессор.
Примечание. Нажатие клавиши Enter выполняет то же действие, что и двойной щелчок.
Одиночный щелчок также выполняет то же действие, что и двойной щелчок, если у вас
есть:
выбрано Одиночный щелчок, чтобы открыть файл на вкладке « Вид » диалогового окна
«Конфигурация», или
включил «Один щелчок, чтобы открыть элемент» в настройках системы, и вы выбрали
« Использовать текущие настройки системной папки» на вкладке « Вид » диалогового
окна «Конфигурация».
